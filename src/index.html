Beginner Roadmaps

Are you an Absolute beginner? Here are some beginner friendly roadmaps you should start with.

Frontend Developer

Develop the part of web apps that users interact with i.e. things rendered in the browser.

Backend Developer

Develop the part hidden from the user e.g. things like APIs, databases, search engines etc.

Full Stack Developer

Develop both the frontend and backend side of the web apps i.e. the whole development stack.

There is also a beginner DevOps roadmap which requires you to have some backend knowledge and entails a lot of operations work i.e. deploying, scaling, monitoring, and maintaining applications.
