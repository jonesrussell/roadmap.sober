## Installation

1. Clone the repo:
	```sh
	git clone https://github.com/jonesrussell/sober.git
	```
2. Navigate to the project directory:
	```sh
	cd sober
	```
3. Install dependencies:
	```sh
    task default
	```

## Usage

1. Run the application:
	```sh
	task run
	```
2. Open your browser and navigate to `http://localhost:8080`
